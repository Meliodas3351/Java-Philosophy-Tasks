Создайте класс Cycle c производными классами Unicycle, Bicycle, Tricycle. Покажите, что объект любого из производных
классов можеть быть преобразован в базовый на примере метода ride().

Добавьте аннотацию @Override в производные классы

В класс Cycle добавьте метод wheels(), который будет выводить кол-во колес у каждого объекта базового класса
и его производных.

В производные классы включите метод balance(). Создать экземпляры всех трех проихводных классов, объединить их
восходящее преобразование в массив Cycle, при помощи нисходящего преобразования попробуйту вызвать метод balance() для
каждого члена массива.
